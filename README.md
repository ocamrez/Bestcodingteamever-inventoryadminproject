Aotearoa Treasures Invetory Management System
C++ inventory management system project

App Structure

-Arrays to store products and employee information
-decimal and integer storage for pricing and quantity stock

COMPLETED FEATURES
 - Product Management
   - View products (shown across all three stores)
   - update product info
   - delete product
   - Add product

 - Low stock alert (below 5 stock)

 - Employee management
   - View employee list
   - update employee info (name + location)
   - delete employee
   - add employee
